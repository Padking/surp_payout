Порядок работы с приложением

1. pip install -r requirements.txt,
2. заменить значение переменной "url_for_webhook" в .action_func во views.py,
3. python manage.py runserver,
4. http://127.0.0.1:8000/persons/<здесь id модели Persons>/  # в интервале от 1 до 102
